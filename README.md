# Customer Segmentation Analysis

## 📌 Overview
This project analyzes customer data to identify different customer segments based on demographics and spending behavior.

## 🎯 Objectives
- Identify who spends more
- Analyze age group spending patterns
- Understand the effect of profession on spending

## 🧹 Data Cleaning
- Handled missing values
- Cleaned categorical columns
- Processed segmentation column (Var_1)

## 📊 Analysis
- Gender vs Spending
- Age Group vs Spending
- Profession vs Spending
- Marital Status vs Spending

## 📈 Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 🚀 Conclusion
Customer segmentation helps businesses understand customer behavior and improve decision-making.
